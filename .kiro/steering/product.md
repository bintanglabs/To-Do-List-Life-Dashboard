# Life Dashboard — Mini Project

## Technical Constraints
- TC-1: HTML, CSS, Vanilla JavaScript only (no frameworks)
- TC-2: Data Storage using browser LocalStorage API
- TC-3: Must work in Chrome, Firefox, Edge, Safari

## Features
- Greeting with time & date
- Pomodoro focus timer (customizable duration)
- To-Do List with LocalStorage (no duplicates)
- Quick Links saved in LocalStorage

## Challenges Implemented
- Custom name greeting
- Changeable Pomodoro duration
- Prevent duplicate tasks
